Bibliothèques du projet

Cette documentation vous présente les différentes bibliothèques du projet avec des liens vers la documentation officielle.

1. TanStack Query

TanStack Query est une bibliothèque qui simplifie la gestion des données côtés clients. Nous l'utilisons pour le fetching, la mise en cache et la synchronisation des données serveur.

Documentation : https://tanstack.com/query/latest

2. BootStrap React

React Bootstrap est une bibliothèque qui simplifie l'utilisation des composants Bootstrap dans une application React. Nous l'utilisons pour intégrer rapidement des éléments d'interface (boutons, formulaires, modales, grilles) tout en respectant les principes de React et en ayant accès à ses bénéfices.

Documentation : https://react-bootstrap.netlify.app/
